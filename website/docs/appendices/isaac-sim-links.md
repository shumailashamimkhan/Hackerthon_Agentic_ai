# Isaac Sim Links & Resources

This page provides links and resources for NVIDIA Isaac Sim, an essential simulation environment for Physical AI and humanoid robotics applications. Isaac Sim enables photorealistic simulation, synthetic data generation, and training of AI systems in a virtual environment.

- [Isaac Sim Release Notes](https://docs.omniverse.nvidia.com/isaacsim/latest/release_notes.html) - Latest updates and known issues
- [Isaac Sim Installation Guide](https://docs.omniverse.nvidia.com/isaacsim/latest/installation-guide/index.html) - Step-by-step installation instructions
